==============================
IMUNA - CONTROLE DE IMUNIZAÇÃO
==============================

Versão 1.0 - 27/05/2021

O IMUNA é um software desenvolvido para o controle da vacinação em um posto de saúde, por exemplo.

Nele existem dois tipos de usuários:

Agente comum - tem acesso à: Dados Gerais e Controle de Vacinação.
Agente admin - tem acesso à: Dados Gerais, Controle de Vacinação, Registro de Agente e Registro de Vacina.

Dados Gerais
Reune todos os dados acumulados pelo sistema:
1. uma tabela com a relação de número de pessoas vacinadas por vacina e lote;
2. número de agentes cadastrados; 
3. número de pessoas vacinadas;
4. número de pessoas vacinadas somente com a primeira dose;
5. vacinas disponíveis (em estoque);
6. total de doses cadastradas.

Controle de Vacinação
Consulta, insere, edita e exclui registros de vacinação.

Registro de Agente
Consulta, insere, edita e exclui agentes no sistema. Apenas agentes cadastrados terão a permissão de fazer o login.

Registro de Vacina
Consulta, insere, edita e exclui vacinas no sistema.

*Para fazer alguma ação na tabela basta clicar em algum item.

É importante salientar que também é feito um controle de número de doses por vacina e lote, que, ao atingir o total,
fica, no Controle de Vacinação, impossível selecioná-la.

O sistema possui um "Agente 1", designado ao administrador do sistema. Este não pode ser excluído.
Não existe um sistema de cadastro na tela de login pois apenas os agentes admin podem criar um cadastro para um 
novo agente contratado.

O software é totalmente intuitivo, ficando como instrução apenas o comando de apertar a tecla "ESC" quando o foco 
está na tabela, para sair do modo editar/excluir e ir para o adicionar/limpar.
